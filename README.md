## Notes

- The source code and corresponding guides for the feature extraction and image perturbation sections can be found in `ivd_degeneration_analysis\`. The feature selection section has been omitted (in the paper, this section was performed in each model method on a fold-by-fold basis and is not included in `ivd_degeneration_analysis\`).

- The source code for the five models can be found in the five folders named after each model. The implementation of the ProtoNAM section borrows from the ProtoNAM source project: https://github.com/gzxiong/ProtoNAM. Simply place the input files (pre- and post-perturbation CSV feature files, the Pfirrmann score table, and the statistical information CSV—optional(this was not used in the paper)) in the root directory of the corresponding folder. The specific paths can be adjusted in the corresponding `config` files. The format of the CSV feature files matches the feature extraction output format in `ivd_degeneration_analysis`, corresponding to `raw.csv` and `pert.csv` in the config respectively; sample formats for the Pfirrmann scoring table and statistics CSV are provided in `ProtoNAM\pfirr_data_example.xlsx` and `ProtoNAM\statistics_example.csv`, respectively. The requirements for the five models are listed in `E:\upload\ProtoNAM\requirements.txt`; simply run `run.py` in each directory to execute the models.
